# 人员Excel表匹配与拆分需求文档
## 1. 需求背景
现有两份包含人员信息的Excel文件（主表、附表），均含人名、身份证号字段，主表额外包含“户籍所在乡镇（街道）”字段。需通过Python实现人员信息匹配、对比及按户籍维度拆分的操作，满足数据筛选与分类查看的需求。

## 2. 核心功能需求
### 2.1 人员信息匹配并生成新表C
- 以**身份证号**为唯一匹配标识（规避人名重复问题），从主表中筛选出与附表中身份证号重合的人员数据。
- 将筛选出的人员数据完整保存为新的Excel文件（表C），保留主表所有列字段。

### 2.2 主表-附表人员对比
- 生成对比表，清晰展示：
  - 附表中存在且主表中匹配到的人员（标注“主表存在”）；
  - 附表中存在但主表中无对应身份证号的人员（标注“主表不存在”）；
- 对比表需保留附表核心字段（人名、身份证号），并增加“主表存在状态”列。

### 2.3 按户籍拆分表C
- 读取表C的“户籍所在乡镇（街道）”列，按不同街道/乡镇名称分组。
- 为每个分组单独生成一个Excel文件，文件名格式为“[户籍所在乡镇（街道）].xlsx”，文件内容为该分组下的所有人员数据。

## 3. 输入输出要求
### 3.1 输入
- 主表Excel文件：支持.xlsx格式，必须包含列名：人名、身份证号、户籍所在乡镇（街道）；
- 附表Excel文件：支持.xlsx格式，必须包含列名：人名、身份证号。

### 3.2 输出
- 表C：Excel文件，命名为“匹配人员表C.xlsx”；
- 对比表：Excel文件，命名为“主表-附表人员对比表.xlsx”；
- 拆分后的户籍分组文件：若干个Excel文件，存放在“按户籍拆分”文件夹下，文件名对应户籍名称。

## 4. 技术实现要求
- 使用Python语言，核心依赖库为pandas（数据处理）、openpyxl（Excel读写）；
- 代码需包含异常处理（如文件不存在、列名缺失、身份证号格式异常等）；
- 代码注释清晰，参数可配置（如输入文件路径、输出路径）。

# Python实现代码
```python
import pandas as pd
import os
from pathlib import Path

# ===================== 配置参数（可根据实际修改） =====================
MAIN_EXCEL_PATH = "主表.xlsx"  # 主表文件路径
SUB_EXCEL_PATH = "附表.xlsx"   # 附表文件路径
OUTPUT_DIR = "./人员数据处理结果"  # 结果输出根目录
# ====================================================================

def init_dir():
    """初始化输出目录，不存在则创建"""
    Path(OUTPUT_DIR).mkdir(exist_ok=True)
    Path(f"{OUTPUT_DIR}/按户籍拆分").mkdir(exist_ok=True)

def check_columns(df, required_cols, file_name):
    """检查DataFrame是否包含必需列，缺失则抛出异常"""
    missing_cols = [col for col in required_cols if col not in df.columns]
    if missing_cols:
        raise ValueError(f"{file_name}缺失必需列：{','.join(missing_cols)}")

def main():
    try:
        # 1. 初始化目录
        init_dir()

        # 2. 读取并校验输入文件
        print("读取主表和附表数据...")
        df_main = pd.read_excel(MAIN_EXCEL_PATH, engine="openpyxl")
        df_sub = pd.read_excel(SUB_EXCEL_PATH, engine="openpyxl")
        
        # 校验必需列
        check_columns(df_main, ["人名", "身份证号", "户籍所在乡镇（街道）"], "主表")
        check_columns(df_sub, ["人名", "身份证号"], "附表")

        # 3. 匹配人员并生成表C
        print("匹配主表与附表人员...")
        # 提取附表身份证号列表
        sub_id_list = df_sub["身份证号"].tolist()
        # 筛选主表中身份证号在附表中的数据
        df_c = df_main[df_main["身份证号"].isin(sub_id_list)].copy()
        # 保存表C
        c_path = f"{OUTPUT_DIR}/匹配人员表C.xlsx"
        df_c.to_excel(c_path, index=False, engine="openpyxl")
        print(f"表C已生成：{c_path}")

        # 4. 生成主表-附表对比表
        print("生成人员对比表...")
        df_compare = df_sub.copy()
        # 判断附表人员是否在主表中
        df_compare["主表存在状态"] = df_compare["身份证号"].apply(
            lambda x: "主表存在" if x in df_main["身份证号"].values else "主表不存在"
        )
        # 保存对比表
        compare_path = f"{OUTPUT_DIR}/主表-附表人员对比表.xlsx"
        df_compare.to_excel(compare_path, index=False, engine="openpyxl")
        print(f"对比表已生成：{compare_path}")

        # 5. 按户籍拆分表C
        print("按户籍拆分表C...")
        # 按户籍分组
        group_by_hj = df_c.groupby("户籍所在乡镇（街道）")
        for hj_name, group_df in group_by_hj:
            # 处理户籍名称中的特殊字符（避免文件名非法）
            safe_hj_name = hj_name.replace("/", "_").replace("\\", "_").replace(":", "_")
            split_path = f"{OUTPUT_DIR}/按户籍拆分/{safe_hj_name}.xlsx"
            group_df.to_excel(split_path, index=False, engine="openpyxl")
            print(f"户籍[{hj_name}]的文件已生成：{split_path}")

        print("所有操作执行完成！")

    except FileNotFoundError as e:
        print(f"错误：文件不存在 - {e}")
    except ValueError as e:
        print(f"错误：数据校验失败 - {e}")
    except Exception as e:
        print(f"未知错误：{e}")

if __name__ == "__main__":
    main()
```

## 代码使用说明
1. 安装依赖库：`pip install pandas openpyxl`；
2. 修改代码中`MAIN_EXCEL_PATH`、`SUB_EXCEL_PATH`为实际的主表/附表文件路径；
3. 运行代码，结果会生成在`人员数据处理结果`文件夹下；
4. 若户籍名称包含特殊字符（如/、\、:），代码会自动替换为下划线，避免文件名非法。
